# Création d'une API

Prérequis :
- NodeJS
- Npm

Installation :

```bash
git clone https://github.com/MarieLBernard44/just_api.git
cd just_api
npm i
```

Démarrage :

```bash
npm start
```

Accès à l'API sur l'url : `http://localhost:3000`
